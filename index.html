ยินดีต้อนรับ

เลือกอุปกรณ์คอมพิวเตอร์จากปุ่มด้านบนเพื่อเรียนรู้รายละเอียด พร้อมภาพและวิดีโอประกอบ

CPU คือสมองของคอมพิวเตอร์ ทำหน้าที่คิด คำนวณ ประมวลผลคำสั่งทุกอย่าง เช่น การเปิดโปรแกรม เล่นเกม หรือทำงานต่าง ๆ ภายในเครื่อง ยิ่ง CPU แรง คอร์เยอะ ความเร็วสูง เครื่องก็ยิ่งลื่น ทำงานได้หลายอย่างพร้อมกันแบบไม่สะดุด เหมือนเป็นหัวหน้าที่คอยสั่งงานอุปกรณ์ทุกชิ้นในคอม

CPU คือสมองของคอมพิวเตอร์ ทำหน้าที่ประมวลผลคำสั่งต่าง ๆ

“RAM คือพื้นที่ที่คอมใช้เก็บข้อมูลชั่วคราวในขณะที่เครื่องกำลังทำงาน เช่น เปิดเบราว์เซอร์ โปรแกรม เกม ถ้า RAM เยอะ เครื่องจะลื่น เปิดหลายโปรแกรมพร้อมกันได้ ถ้า RAM น้อยจะช้า หน่วง หรือค้าง เพราะไม่มีพื้นที่เพียงพอให้ระบบโหลดข้อมูล

RAM ใช้เก็บข้อมูลชั่วคราวเพื่อให้ CPU เข้าถึงได้รวดเร็ว

เป็นที่เก็บข้อมูลหลักของคอม เช่น รูปภาพ วิดีโอ ไฟล์งาน โปรแกรม และระบบ Windows • HDD – ความจุเยอะ ราคาถูก แต่ช้า • SSD – เร็วมาก เปิดเครื่องไว โหลดเกมเร็ว ทำให้ระบบทำงานลื่นกว่า คอมยุคใหม่เกือบทุกเครื่องใช้ SSD เป็นหลักเพราะความเร็วสูงกว่าหลายเท่า

อุปกรณ์เก็บข้อมูลถาวร เช่น ไฟล์งาน ระบบปฏิบัติการ และโปรแกรม

เมนบอร์ดคือชิ้นส่วนที่เชื่อมต่ออุปกรณ์ทุกอย่างในคอมเข้าด้วยกัน เช่น CPU, RAM, SSD, การ์ดจอ, พอร์ตต่าง ๆ ทำหน้าที่ให้ทุกชิ้นทำงานร่วมกันได้อย่างสมบูรณ์ เปรียบเหมือนร่างกายที่มีเส้นเลือดเชื่อมอวัยวะทั้งหมด ถ้าเมนบอร์ดดี จะรองรับการอัปเกรดและเพิ่มอุปกรณ์ในอนาคตได้ง่าย

Mainboard เชื่อมต่ออุปกรณ์ทั้งหมดให้ทำงานร่วมกันได้
